# Calculator
```
Scientific Calculator
```
## Getting Started
### Software dependencies
- .NET 6 SDK
- Node v16.18.x
- Dotnet CLI

## Frontend
- In your terminal
  - `cd ./src/client`
  - `npm install` to install the node modules
  - `npm start` to start the frontend in your local browser

## Backend
- In your terminal
  -  `cd ./src/server/`
  - `dotnet run`
